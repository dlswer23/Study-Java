# 06 조건문



#### 조건문이란

상황에 따라 흐름을 결정하는 구문이다.



#### 조건문의 종류 

+ if 문 : 특정 행동을 할 것인지 안할 것인지 결정



+ If-else문 : 둘 중 하나를 선택 할 때 

​      참이면 'a'를 실행, 거짓이면 'b'를 실행 등



+ else-if문 : 이는 여러가지 선택지 중에 하나를 선택

  성적이 90이상이면 'A', 있는 80이상이면 'B'등 

예시 ) if ( 90점 보다 큼 ? ){

A학점 - 취소

}else if( 80점보다 큼?){

B학점

}else {
