# 04 타입과 형변환



#### 타입관련 주의점



1.나눗셈 주의

int 와 int 를 나누게 된다면 당연히 결과값도 int 입니다

double a = 5.0 / 2.0;

int b = 4/2;

int c = 5/2;

이라고 할때 과연 c의 갑은 무엇인가?

답은 2.5가 아닌 2입니다. 정수끼리 나누었기 때문에 값도 당연히 int가 됩니다



2.타입 불일치 주의

값과 대입하려는 타입이 맞지 않으면 안된다, 



//예

String seven = 7; // ERROR

String seven = "7" ; //OK

int seven = 7;  // OK
